Automated AI Twitch Chatbot

A hobby project built to test whether a full end-to-end AI streaming assistant could be made practical, controllable, and voice-driven.

Project Overview

Live Streaming AI Assistant Pipeline

Role: Designer and Builder

I developed an automated AI chatbot system for Twitch that integrates API-driven chat interaction, custom voice output, and model orchestration across both local and hosted LLMs. The goal was to build a working system from scratch as a hobby challenge and prove that a reliable, feature-rich setup was possible.

Implementation

Key Components

  • Integrated Twitch Developer API keys for real-time chat ingestion and response flow control.
  • Implemented voice modulation/output using the ElevenLabs API with a custom-trained voice profile.
  • Used fine-tuned OpenAI models alongside a local fine-tuned Llama model for flexible response generation.
  • Built a custom training dataset of 100,000+ synthesized JSONL-formatted samples.
  • Created a GUI that ties together orchestration features, configuration controls, and runtime interaction.
